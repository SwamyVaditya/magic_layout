Initialization

Step 1. Include files

First, we need to include the file animate.css in the head section. Described in the file CSS rules for animations of plugin. Next connect the JS file before </body>:

  • - Jquery library(jquery.js)
  • - Magic Layout plugin(magicLayout.js)

As a result we must receive the following code

					
<head>
 <link rel="stylesheet" href="core/css/css/animate.css" type="text/css"/>
</head>
<body>			
 <script src="core/jquery.js"></script>
 <script src="core/magicLayout.js"></script>
</body>

Step 2. Create HTML-markup

To work on Magic Layout, you must add the .layer to the element that will be animated. The next step is install the settings of plugin with helps data attributes data-show-effect (for the name of effect) and a data-show-delay (to install a delay before starting the animation).

					
<div id="english_portfolio" class="dt_container dt_container_columns3 projects">	
 <a href="#" class="demo_stas_project layer" data-show-effect="zoomIn" data-show-delay="400">
  <h3 class="project_item">Responsive modal window</h3>
 </a>
 <a href="#" class="demo_stas_project layer" data-show-effect="zoomIn" data-show-delay="600">
  <h3 class="demo_stas_project_item">Responsive modal window</h3>
 </a>
 <a href="#" class="demo_stas_project layer" data-show-effect="zoomIn" data-show-delay="800" data-time="3500">
  <h3 class="demo_stas_project_item">Responsive modal window</h3>
 </a>
</div>

Step 3. JS initialization code

					
(function($){
 $('.demo_button').on('click',function(){
  $(this).magicLayout({
   container: $('#english_portfolio')
  });
 });
})(jQuery);

Parameter Description

data-show-delay Delay parameter. Default value - 0 ms
data-time Duration parameter. Default value - 1000 ms.
