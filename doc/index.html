Get Started

Step 1. Include files

First, we need to include the file magicLayout-all.css in the head section. Described in the file CSS rules for animations of plugin. Next connect the JS file before </body>:

  • jQuery
  • Magic Layout plugin - jquery.magicLayout.js

As a result we must receive the following code

						
<head>
 <link rel="stylesheet" href="magicLayout/magicLayout-all.css">
</head>
<body>		

 <!-- Content -->
 
 <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.2/jquery.min.js"></script>
 <script src="magicLayout/jquery.magicLayout.js"></script>
</body>

Step 2. Create HTML-markup

To work on Magic Layout, you must add the js-ml-layer to the element that will be animated. The next step is install the settings of plugin with helps data attributes data-show-effect (for the name of effect), a data-show-delay (to install a delay before starting the animation) and data-duration (for duration of animation). 

						
<button class="button button_white js-start-demo">Start demo</button>

<div class="demo-boxs">
 <div class="demo-box js-ml-layer" data-show-effect="zoomIn" data-show-delay="400"></div>
 <div class="demo-box js-ml-layer" data-show-effect="zoomIn" data-show-delay="600"></div>
 <div class="demo-box js-ml-layer" data-show-effect="zoomIn" data-show-delay="800" data-duration="2000"></div>
</div>

Step 3. JS initialization code

						
$(".js-start-demo").on("click", function(){	
 $(this).magicLayout({
  container: $(this).next()	
 });
});

Settings

Parameter Description
data-show-effect Effects setting (required)
data-show-effect Delay parameter. Default value - 0 ms
data-duration Duration parameter. Default value - 1000 ms

Methods

Name Description
Init

Initialization method

Params: options.container - is parent element for js-ml-layer.

Return: options.container

Destroy

Method for the removal of the class and the animation settings of js-ml-layer.

Params: magicLayoutActive - is active parent element for js-ml-layer.

Return: this

Init

						
$(this).magicLayout({
 container: magicLayout
});

Destroy

						
$(this).magicLayout("destroy", magicLayoutActive);
